สมุนไพรไทยเสี่ยงสูญพันธุ์ 22 ชนิด
ประกาศคุ้มครองและควบคุม 7 ชนิด


1. ถั่วดินโคก



  สรรพคุณ : ลดไข้ในเด็ก แก้พิษงู แก้ฝี บำรุงน้ำนมสตรี


2. เทพทาโร



  สรรพคุณ : แก้ปวดท้อง แก้วิงเวียน แก้อาเจียน แก้หอบหืด


3. มะตูมนิ่ม



  สรรพคุณ : ขับลมในลำไส้ ช่วยให้เจริญอาหาร ช่วยให้นอนหลับ


4. มะหาด



  สรรพคุณ : ใช้ถ่ายพยาธิ แก้อาการคัน


5. เร่ว



  สรรพคุณ : แก้ไข้ แก้ริดสีดวง แก้หืดไอ


6. หัวร้อยรู



  สรรพคุณ : บำรุงหัวใจ ขับชีพจร ขับพยาธิ แก้พิษในข้อในกระดูก


7. กระทุ่มนา



  สรรพคุณ : ช่วยลดความดันโลหิต รักษาโรคผิวหนังทุกชนิด


8. ขันทองพยาบาท



  สรรพคุณ : แก้โรคผิวหนัง แก้ลมพิษ แก้ประดง แก้พิษในกระดูก


9. จุกโรหินี



  สรรพคุณ : ลดระดับน้ำตาลในเลือด แก้อ่อนเพลีย บำรุงกำลัง


10. ชะเอมไทย



  สรรพคุณ : แก้โรคในลำคอ แก้ลม แก้เลือดออกตามไรฟัน


11. ชิงชี่



  สรรพคุณ : แก้โรคที่เกิดในท้อง ขับลมภายใน แก้ไข้ร้อนในทุกชนิด


12. ตับเต่า



  สรรพคุณ : ใช้ดับพิษร้อน แก้ไข้ แก้ร้อนใน


13. นางแย้มป่า



  สรรพคุณ : แก้ไข้ รักษาลำไส้อักเสบ ขับปัสสาวะ


14. ปลาไหลเผือก



  สรรพคุณ : ถ่ายพิษต่างๆ แก้ปวดท้อง


15. พังคี



  สรรพคุณ : แก้ไข้ แก้ท้องอืด แก้จุกเสียด


16. มะคังแดง



  สรรพคุณ : แก้ปวดท้อง แก้ปวดเส้นเอ็น


17. สะค้าน



  สรรพคุณ : ขับลมในลำไส้ แก้แน่น แก้จุกเสียด


18. สารภีป่า



  สรรพคุณ : บำรุงหัวใจ แก้ไข้ ขับลม


19. อบเชยไทย



  สรรพคุณ : แก้อ่อนเพลีย ขับลม บำรุงธาตุ


20. เฉียงพร้านางแอ



  สรรพคุณ : แก้ร้อนใน รักษาไข้ ขับลม


21. เถาเอ็นอ่อน



  สรรพคุณ : แก้ปวดเมื่อยเส้นเอ็น แก้ขัดยอก


22. เปราะหอม



  สรรพคุณ : แก้หวัดคัดจมูก แก้เสมหะ
